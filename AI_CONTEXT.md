# AI Context — 임베디드 TIL

이 파일을 대화 시작 전에 붙여넣으면 맥락 파악 가능.

---

## 나는 누구

- iOS 개발 경험 있는 주니어 개발자, 임베디드 SW로 전환 중
- 목표: STM32 bare-metal 포트폴리오 완성 → 임베디드/전장 취업

---

## 이 저장소 목적

임베디드 공부하면서 배운 것 마크다운으로 기록. 코드 없음, 글만.

---

## 학습 순서

```
1. C 언어 (포인터, 비트 연산, volatile, 구조체)
2. STM32 GPIO (MODER, ODR, IDR 레지스터 직접 제어)
3. STM32 UART (printf 리다이렉트, 디버깅)
4. STM32 I2C (레지스터 직접, HAL 미사용)
5. STM32 CAN (루프백 테스트)
```

---

## 폴더 구조

TIL/
