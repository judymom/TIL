## 작성 규칙

- 대제목은 `# 주제명` 형식으로 작성한다.
- 주요 섹션은 `## 1. 제목` 형식으로 번호를 붙인다.
- 하위 개념은 필요할 때만 `### 제목`을 사용한다.
- 기본 콜아웃은 라벨 없이 사용한다.
- 실수하기 쉬운 내용과 예외만 `> **주의**`, `> **예외**` 라벨을 사용한다.
- 목록은 기본적으로 `-`를 사용한다.
- 순서가 중요한 내용만 `1.` 형식의 번호 목록을 사용한다.
- 섹션 사이는 `---`로 구분한다.
- 마지막에는 `## 참고 자료` 섹션을 둔다.
