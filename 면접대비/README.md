# 면접대비

학습 노트와 분리된 면접 전용 자료 모음. 짧고 입에 붙이는 용도.

---

## 폴더 구성

```
면접대비/
├── README.md          ← 이 파일
├── _오답노트.md       ← 퀴즈 중 모호하게 답한 용어 자동 누적
├── 답변카드/          ← 개념별 30초 면접답변
│   ├── 포인터.md
│   ├── volatile.md
│   └── ...
└── 예상질문.md        ← 임베디드/C 빈출 질문 리스트
```

---

## 사용 흐름

1. `c-lang/`, `stm32/`, `cs/` 에 **학습 노트** 작성 (길고 상세하게)
2. `/make-card <노트경로>` 실행 → 학습 노트에서 **답변카드 자동 추출**
3. 추출된 답변카드 검토/보완
4. `/easy-quiz` 로 입에 붙이기 연습

---

## 학습 노트 vs 답변카드

| 학습 노트 | 답변카드 |
|----------|---------|
| `c-lang/포인터.md` | `면접대비/답변카드/포인터.md` |
| 길고 상세 (코드, 예외, 심화) | 30초 분량 핵심만 |
| 천천히 읽고 이해용 | 입으로 굴리는 암기용 |
| 작성 = 머리로 이해 | 작성 = 입에 붙이기 |

같은 개념을 2번 쓰는 건 의도된 설계입니다. 한 번은 머리로, 한 번은 입으로.

---

## 명령어

| 명령어 | 동작 |
|--------|-----|
| `/make-card <파일>` | 학습 노트에서 답변카드 N개 추출 |
| `/easy-quiz` | 무작위 답변카드 기반 부드러운 면접 |
| `/easy-quiz <주제>` | 특정 주제 면접 |
| `/easy-quiz 오답` | `_오답노트.md` 기반 미니 퀴즈 |
