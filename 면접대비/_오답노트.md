# 오답노트

`/easy-quiz` 실행 중 사용자가 모호하게 답하거나 못 떠올린 정확한 용어를 자동으로 축적합니다.
다시 마주칠수록 빈도가 올라가고, 자연스럽게 입에 붙습니다. 외우려 하지 않아도 됩니다.

---

## 자주 놓치는 정확 표현

| 모호한 표현 | 정확한 용어 | 출처 파일 | 빈도 |
|------------|-----------|----------|------|
| _(예시)_ "값 안 바뀜" | "컴파일러 최적화로 레지스터 캐싱" | c-lang/volatile.md | 0 |

---

## 사용 방법

- `/easy-quiz` 시 자동 갱신됨
- 직접 수정 가능
- 빈도 3 이상 = 다음 퀴즈에서 우선 출제
- `/easy-quiz 오답` 으로 이 노트 기반 미니 퀴즈

---

## 작성 규칙

- 사용자 답을 그대로 기록 (창피주기 X, 패턴 인식 O)
- 정확한 용어는 학습 노트 본문에서 추출
- 같은 모호 표현 중복 시 빈도 +1, 새 행 추가 X
